# Mesh — Decentralized AI

Futuristic AI Operating System landing page. Premium, minimal, terminal-flavored — Linear/Vercel/Warp/OpenAI inspired.

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion

## Run
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
```

## Sections
`components/`
- `Nav` — sticky glass navbar
- `Hero` — animated Mesh logo, "One mesh. Every model."
- `TerminalDemo` — live typing + checklist (DeepSeek R1 → reports)
- `FeatureGrid` — Chat, Coding, Research, Image, Video, Folder Automation, Finance Copilot
- `AIRouter` — animated model routing visualization
- `FinanceOS` — forecasting, budgeting, variance, commissions, KPI dashboard mock
- `Footer` — CTA + links

## Theme
Obsidian black bg, glassmorphism cards, emerald (`#34f5a0`) + soft blue (`#5e9dff`) accents, white type. Tokens in `tailwind.config.ts`, utilities in `app/globals.css`.
